Minecraft

Minecraft is one of my all-time favorite games. Throughout my time playing the game, I've also gained experience with its built-in modding tools, resource packs and data packs. Minecraft is also what let me to begin learning Java in order to create plugins for Spigot, a modified Minecraft server that provides API access to the Minecraft server.

I am an administrator on a small Minecraft server called Wings of Fire: The New World, which is based on Tui T. Sutherland's Wings of Fire series. My largest Java project is Dreamvisitor, an all-in-one utility plugin for the server. However, I also have publicly available projects, most of which are features extracted from Dreamvisitor. Below are some notable projects.

The links and stats here are from Modrinth, but my plugins are also available on the SpigotMC forum and the Paper Hanger.

Compass Tracker

A manhunt data pack that allows any number of hunters to track any number of runners in all three dimensions.

No Creeper Craters

A very lightweight plugin that prevents Creepers from breaking blocks.

Toasty!

This data pack adds Toasty, a companion that will follow you throughout the world of Minecraft!

Pause Chat

A Spigot plugin that temporarily (or permanently) disables Minecraft chat.

ItemVoid

A Bukkit plugin that removes blacklisted items from players.

Ring of Fire

A simple asymmetrical multiplayer Minecraft minigame for a variable number of players.

PlayerCap

Override the player limit!

Wings of Fire: The New World

Wings of Fire: The New World is a hybrid Discord-Minecraft server started in 2016. Throughout its life, the server has seen four map revisions and has garnered a community of over 1,500 members. The server's story is set thousands of years following the canon events of Wings of Fire and boasts a unique cast of characters and an extensive new world full of lore to explore!
